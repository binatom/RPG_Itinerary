string Spells::print_categories ( )
virtual

A normal member returning string.

The method executes sql command and checks the correct execution. Method uses the sqlite3_exec() interface, which is a convenience wrapper around sqlite3_prepare_v2(), sqlite3_step(), and sqlite3_finalize(). Third argument of sqlite3_exec() is call_back1 function, which is define in abstract class Equipment. The call_back1 function is printing spells categories.
